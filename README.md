Root locale standard del repo: `C:\baubook`.


## Comandi base

```powershell
cd C:\baubook
.\scripts\install-clean.ps1
npm run docs:check
npm run launch:check
npm run typecheck
.\baubook.ps1 -Mode web
.\baubook.ps1 -Mode android-build -CleanPrebuild
.\baubook.ps1 -Mode android-dev
```
